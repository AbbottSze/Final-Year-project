LFSR Description

An Linear Feedback Shift Register (LFSR) is a shift register that uses a linear function (usually XOR) of previous bits to generate new bits. It's commonly used in cryptography, random number generation, and error checking.

How it works (Steps):

1. Start with an initial binary value (called the seed).

2. Shift all bits to the right by one position.

3. Calculate a new bit using XOR on selected bits (called taps).

4. Insert the new bit at the leftmost position.

5. Repeat the process for each new bit.

CBC Description

CBC is a mode of encryption that chains blocks together. Each plaintext block is XORed with the previous ciphertext block before encryption, making patterns harder to detect.

How It Works:

1. Start with an Initialization Vector (IV) — a random block used for the first step.

2. XOR the first plaintext block with the IV.

3. Use the output as the "chaining value" for the next block.

4. Repeat: XOR the next plaintext block with the previous ciphertext.
